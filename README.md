Preto&Branco Motel - Landing Page
Esta é uma landing page moderna e responsiva desenvolvida para o Preto&Branco Motel, um estabelecimento tradicional e pioneiro na Região Sudoeste de Goiânia. O foco do projeto foi criar uma interface sofisticada que une a tradição da marca com a praticidade digital.

Objetivo do Projeto
Proporcionar uma experiência de usuário (UX) fluida, facilitando o acesso ao tour virtual, galeria de suítes e reserva direta via WhatsApp.

Tecnologias Utilizadas
HTML5: Estruturação semântica para melhor SEO.

CSS3: Design minimalista P&B, Glassmorphism e layout responsivo.

JavaScript: Lógica de interatividade e manipulação de DOM.

Git & GitHub: Versionamento e deploy.

Destaques Técnicos
UI/UX Personalizado: Botões com design arredondado (pílula) e tipografia sofisticada.

Mobile First: Totalmente adaptado para dispositivos móveis.

Integração Externa: Links dinâmicos para Tour 360º e API do WhatsApp.
